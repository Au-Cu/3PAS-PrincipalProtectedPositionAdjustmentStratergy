# 3PAS：Principal-Protected Position Adjustment Strategy

基于本金保护的动态仓位调整策略

---

## 项目简介

3PAS（Principal-Protected Position Adjustment Strategy）是一种**风险控制导向的稳健型动态资产配置策略**。
不同于固定仓位投资，该策略基于本金保护机制，根据市场变化动态调整仓位，使在**尽可能保住预设保护本金**的基础上，**获得较为可观的收益**。
经过模拟实盘回测，策略在多种不同的市场环境和投资标的组合下均能保证：

- **总资产不跌破保护本金 (protected)**

- **在保证安全边界的前提下提高资金利用效率**

策略为投资者提供了一个明确的风险边界与仓位调整机制。

---

## 核心思想

将总资产划分为两部分：

| 成分 | 含义 |
|:---|:---|
| 保护本金 | 不允许发生亏损的资产（总资产的止损线，投资终止的吸收壁） |
| 绝对亏损容忍度 | 可承受全部亏损的风险资产 |

- 保护本金的额度由投资者在使用本策略投资前，视自身情况自行设定。*一经设定，不再更改*
- 绝对亏损容忍度会随着持仓市值的变动而不断波动

通过**在每天开盘集中竞价时根据最新资产情况调整仓位**，使得即使出现最大允许跌幅，也不会跌破保护本金。

---

## 变量定义

| 变量名 | 符号 | 备注 |
|:---|:---:|:---|
| 第 $n$ 天开盘时的总资产 | $total_n$ | 单位：元；初始本金： $total_1$ |
| 保护本金 | $protected$ | 单位：元 |
| 单日相对亏损容忍度 | $tolerance$ |  |
| 第 $n$ 天绝对亏损容忍度 | $total_n - protected$ | 单位：元；亏损容忍本金：  $total_1 - protected$ |
| 第 $n$ 天开盘时调仓前的持仓市值 | $market_n$ | 单位：元 |
| 第 $n$ 天开盘时调仓后的持仓市值 | $market_n^*$ | 单位：元 |
| 第 $n$ 天开盘时的空仓资金 | $total_n-market_n$ | 单位：元 |
| 第 $n$ 天的股票涨跌幅 | $change_n$ |  |
| 股票涨跌停幅度 | $limit$ |  |

---

## 基本原理

策略基于上一交易日股价变化况动态调整仓位，根据策略给出的当日目标持仓市值进行买卖操作调整，使得即使股票出现最大容忍跌幅，账户资产仍不会跌破保护本金 $protected$ 。即第 $n$ 天至多可能发生 $\min(tolerance,limit)$ 的相对回撤，或 $\min(protected ,market_n \times limit)$ 元的绝对回撤。 
**第 $n$ 天的目标持仓市值 $market_n^*$ 由以下公式给出：**

$$
market_n^* = \min\left(\frac{total_n - protected}{tolerance}, total_n \right)
$$

---

## 仓位调整规则

只在每天开盘集合竞价时根据上一日股价变化进行一次买或卖操作。

### 首日建仓

第1天开盘时买入：

$$
market_1 = \frac{total_1 - absol}{tolerance}
$$  

### 第 $n$ 天调仓

#### 止损条件
若：

$$
total_n < protected
$$

则**立即终止策略，清仓离场**；否则，按照以下公式，通过买卖操作将持仓市值调整到目标持仓市值：  

第 $n$ 天开盘时的目标持仓市值为：  

$$
market_n^* = \frac{total_n - protected}{tolerance}
= \frac{total_{n-1} + market_{n-1} \times change_{n-1} - protected}{tolerance}
$$

## 参数选择

首次建仓时，投资者需根据自身情况，选择合适的策略参数。
参数一经选定，不再允许更改变动。

### 保护本金 $protected$

投资者需将初始本金划分为保护本金 $protected$ 和亏损容忍本金 $total_n - protected$ 两部分。
保护本金额度的选择包括但不限于以下情况：

- 普通的止损线，即**初始本金减去亏损容忍本金**
- 初始本金中**融资贷款的杠杆资金**
- 初始本金中**一段时间后需要提现使用的目标存款**
- 初始本金中**用于防范风险，未知具体使用时间的长期存款**

### 相对亏损容忍度 $tolerance$

#### 默认设置
从容错性出发考虑，一般建议设置相对亏损容忍度 $tolerance$ **略高于股票涨跌停幅度 $limit$ （如1.5倍左右）** 。这使得账户可以在小概率的极端行情（跌停）中存活下来，为起死回生、绝地反击留有一定仓位，而不至于一夜之间连本带利亏损，甚至触及保护本金而被迫清仓离场。  

#### 稳健设置
若**希望降低风险，加强本金保护，可以适当提高 $tolerance$**。
适用场景：

- **追求稳健**
- **在牛市中做左侧交易**，在市场过热时防守牛市见顶转熊，**追求保住已有收益**

#### 激进设置

若**希望以承担一定保护本金亏损风险以搏取更大收益，可以适当降低 $tolerance$**。
适用场景：

- **牛市主升浪阶段**
- **在熊市中做左侧交易，埋伏底部反转**

## 展望

### 应用场景

#### 杠杆投资本金保护
**将融资贷款的杠杆资金设置为保护本金**，自有资金设置为绝对亏损容忍度，则可**在保证融资资金安全的前提下获得杠杆收益，极大降低融资风险**。  

#### 存款入市投资
适用于：

- 目标存款，即一段时间后需要提现使用的存款，如**购房、购车、子女教育储备资金**等
- 长期存款，或用于防范风险的存款，如**养老资金、医疗储备资金**等

将以上**存款暂时挪用于稳健投资**，配合手头的闲置资金作为亏损缓冲，可在**保障本金安全的前提下参与市场投资，抓住增值机会**。  

#### 情绪管理与投资纪律
由**投资者事先自行决定亏损上限**，可以极大限度上：

- **减少非理性的情绪化操作、恐慌性交易**
- **降低不确定性，为投资者提供掌控感和安全感**
- **提高投资纪律**
- **避免洗盘急跌中恐慌割肉**的情况

### 存在的不足

- 未考虑流动性枯竭的极端情况，如连续涨停或连续跌停
- 未考虑开盘竞价涨跌幅
- 简化认为可以以上一日收盘价成交
- 未考虑交易成本，如印花税、交易佣金等
- 暂时只支持持仓单只股票的情况
这些因素在实盘交易中可能会对策略表现产生一定影响。
