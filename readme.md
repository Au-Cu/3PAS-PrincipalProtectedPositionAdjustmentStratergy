# 变量符号设定

| 变量名 | 符号 | 备注 |
|--------|------|------|
| 第n天开盘时的总资产 | $total_n$ | 单位：元；初始本金： $total_1$ |
| 保护本金 | $protected$ | 单位：元 |
| 单日相对亏损容忍度 | $tolerance\%$ |  |
| 第n天绝对亏损容忍度 |  | $total_n-protected$；单位：元 |
| 第n天开盘时未调仓的股票市值 | $market_n$ | 单位：元 |
| 第n天开盘时调仓后的股票市值 | $market_n^*$ | 单位：元 |
| 第n天开盘时的空仓资金 |  | $total_n-market_n$ |
| 第n天的股票涨跌幅 | $change_n%$ |  |
| 股票涨跌停幅度 | $limit\%$ |  |

---

# 策略

## 核心思想

基于上一交易日股价变化的最新情况动态调整仓位，使得当天的最大回撤尽可能不亏损到保护本金 $protected$，即第n天至多允许 $\min(tolerance\%,limit\%)$ 的相对回撤，或 $\min\left(protected,market_n \times limit\%\right)$ 元的绝对回撤。$market_n^*$ 始终满足  

$$
market_n^* = \min\left(\frac{total_n - protected}{tolerance\%}, total_n \right)
$$

---

## 相对亏损容忍度选择

一般情况下，默认相对亏损容忍度 $tolerance\%$ 即为股票涨跌停幅度 $limit\%$ 。这能使得在最坏情况下，当日亏损也不触及保护本金 $protected$ 。  

从容错性出发考虑，一般建议设置相对亏损容忍度 $tolerance\%$ 为1.5倍左右的涨跌停幅度 $limit\%$ ，这使得账户可以在小概率的极端行情（跌停）中存活下来，为起死回生、绝地反击留有一定仓位，而不至于利润一夜归零，甚至触及保护本金而被迫清仓离场。  

愿意承担一定保护本金亏损风险以搏取更大收益的，如在牛市主升浪阶段，或是在熊市中做左侧交易，埋伏底部反转的投资者可以适当减小 $tolerance\%$ 的值，将更多空仓的本金转入市场中。  

希望绝对保住保护本金，追求稳健的，或是在牛市中做左侧交易、追求保住已有收益，在市场过热时防守牛市见顶转熊的交易者，可以适当增大 $tolerance\%$ 的值，降低仓位。  

---

# 操作

只在每天开盘竞价时根据上一日股价变化进行一次买或卖操作。若当天开盘时的总资产 $total_n$ 跌破保护本金 $protected$，则立即止损，清仓卖出所有股票；否则，按照以下公式，通过买卖操作将股票市值调整到目标市值：  

第1天开盘时：  

$$
market_0 = \frac{total_0 - absol}{tolerance\%}
$$  

第n天开盘时的市值调仓目标：  

$$
market_n^* = \frac{total_n - protected}{tolerance\%}
= \frac{total_{n-1} + market_{n-1} \times change_n\% - protected}{tolerance\%}
$$

# 展望

## 策略实用价值

- 杠杆投资保护本金：将融资借贷所得资金设置为保护本金 $protected$ ，自己的闲置资金设置为绝对亏损容忍度，则可在保证融资资金绝对安全的前提下，实现杠杆收益，极大降低融资风险。  

- 长期存款投资：将用于养老、购房、教育、医疗等的风险防范存款和目标存款暂时挪用于稳健投资，保障存款安全的前提下参与市场，抓住增值机会。  

- 情绪缓冲与投资纪律：明确给出“最多也就亏多少”，为投资者提供心理安全边界；亏损上限由投资者事先自行决定，极大降低不确定性，为投资者提供掌控感和安全感；减少非理性情绪化交易，极大限度上降低在洗盘急跌中恐慌割肉的情况。  

---

## 策略存在的不足

- 未考虑开盘竞价涨跌幅，简化认为能以上一日收盘价买入或卖出。  
- 未考虑流动性枯竭的极端情况，如连续涨停或连续跌停。
